# Ejemplo de diseño: Dossier "ENAE & China"

Pieza **oficial** de ENAE International Business School usada como referencia
canónica del **registro editorial premium** de la marca.

## Contenido
- `Dossier_ENAE_China.pdf` — el dossier completo (20 páginas, A4 vertical).
- `paginas/p-01.png … p-20.png` — cada página renderizada como imagen.

## Cómo usarlo
Antes de maquetar cualquier dossier, brochure, portada o pieza de campaña, **abre
este ejemplo** y observa la maquetación real. La guía completa de cómo replicar
este estilo está en `references/dossier.md`.

## Páginas de referencia rápida
| Página | Qué demuestra |
|--------|---------------|
| p-01 | Portada: foto a sangre + overlay gradiente granate + título mixto |
| p-02 / p-03 | Fondo granate pleno, números fantasma, tarjetas translúcidas |
| p-06 / p-14 | KPIs gigantes, gráficos, fotos a sangre |
| p-15 / p-19 | Grids de tarjetas blancas sobre fondo oscuro/granate |
| p-16 | Fondo negro + watermark "E" + KPIs editoriales |
| p-20 | Contraportada de contacto: logo blanco centrado |

## Decisiones de marca que confirma este ejemplo
- El **título mixto** (sans bold + serif itálica + sans light) es la firma visual.
- ENAE usa **fondos oscuros y gradientes** en el registro editorial (no solo blanco).
- La **fotografía va siempre a sangre** y con lavado granate.
- El color de marca sigue siendo el **granate `#a91831`** — nunca azul.
